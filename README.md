# Blog API

Бэкенд для мобильного приложения блога с админ-панелью. 

## Стек

- **PHP 8.3**
- **Laravel 13**
- **MySQL 8.0**
- **Laravel Sanctum** (для API-токенов)
- **Laravel Orchid** (для админ-панели)
- **Postman** (для тестирования API)

## Эндпоинты
| Метод | URL            | Описание          | Доступ   |
|--------|----------------|-------------------|-----------|
| POST   | `/api/register`| Регистрация       | Публичный |
| POST   | `/api/login`   | Авторизация       | Публичный |
| POST   | `/api/posts`   | Создание поста    | Приватный |
| GET    | `/api/posts`   | Все посты         | Приватный |
| GET    | `/api/posts/my`| Мои посты         | Приватный |

## Роуты админ-панели
| URL                      | Описание                  | Доступ                  |
|---------------------------|----------------------------|--------------------------|
| `/admin`                  | Главная страница           | Только авторизованные    |
| `/admin/login`            | Страница входа             | Публичный                |
| `/admin/logout`           | Выход из системы           | Только авторизованные    |
| `/admin/users`            | Список пользователей       | Только авторизованные    |
| `/admin/users/create`     | Создание пользователя      | Только авторизованные    |
| `/admin/users/{id}/edit`  | Редактирование пользователя| Только авторизованные    |
| `/admin/posts`            | Список публикаций          | Только авторизованные    |
| `/admin/posts/create`     | Создание публикации        | Только авторизованные    |
| `/admin/posts/{id}/edit`  | Редактирование публикации  | Только авторизованные    |
